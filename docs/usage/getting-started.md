# Getting Started

This section helps you quickly set up and run a semantic release for your PowerShell project using **PSSemanticRelease**.

- [Install](./installation.md) **PSSemanticRelease** in your project
- Configure your Continuous Integration service to [run `Invoke-SemanticRelease`](./ci-configuration.md)
- Configure your Git repository and package manager repository authentication in your Continuous Integration service
- Configure **PSSemanticRelease** options and plugins
